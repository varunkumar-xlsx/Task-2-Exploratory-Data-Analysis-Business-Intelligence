# Task-2 EDA & BI

## Project Overview
This project focuses on performing Exploratory Data Analysis(EDA) on the superstore sales dataset.

## Objectives
- Analyze total sales
- Identify top performing products
- Analyze sales by category and region
- Study monthly sales trends

## Tools used
- Python (Pandas,Matplotlib)
- MYSQL
- Google Collab

## Key Analysis Performed
- Total sales calculation
- Top 5 products by sales
- Sales by Category
- Monthly sales Trend
- Sales by Region

## Outcome
The Analysis identified key revenue-driving products , top performing regions, and overall sales patterns.
